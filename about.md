# About

Hospitalist Ops is a blog exploring what happens when we bring rigorous probabilistic reasoning to the bedside.

Medicine's diagnostic paradigm was designed for an era when all computation had to happen between two ears. We now have the data, the math, and the tools to decompress that reasoning — to make diagnosis transparent, sequential, and quantitative.

This blog is written by **Akash Venkataramanan**, a hospitalist building open-source tools for Bayesian clinical reasoning using real hospital data.
